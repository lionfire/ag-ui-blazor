# Release Notes

## Version 0.1.0-preview.1

**Release Date**: TBD

This is the initial preview release of LionFire.AgUi.Blazor, providing Blazor components for AI agent chat interfaces.

### Packages Included

- **LionFire.AgUi.Blazor** (0.1.0-preview.1) - Core abstractions and interfaces
- **LionFire.AgUi.Blazor.MudBlazor** (0.1.0-preview.1) - MudBlazor UI components
- **LionFire.AgUi.Blazor.Server** (0.1.0-preview.1) - Blazor Server optimizations
- **LionFire.AgUi.Blazor.Wasm** (0.1.0-preview.1) - WebAssembly client support
- **LionFire.AgUi.Blazor.Testing** (0.1.0-preview.1) - Test utilities

### Features

- Initial project structure and package configuration
- Core abstractions for AG-UI integration
- MudBlazor-based chat components
- Support for both .NET 8.0 and .NET 9.0
- Symbol packages for debugging

### Known Issues

- This is an early preview release; APIs may change significantly
- Documentation is incomplete

### Breaking Changes

N/A - Initial release
